building direction:
cd network, make, make install; then you will get  libnetwork.so in Giraffe directory.
back to  Giraffe directory, then  make base; make giraffe;  make client; after compiling, in bin directory, you will get executable files for Giraff server and client.
Modify bin/Giraffe_conf/ configurables, then execute./giraffe ./giraffeclient for Giraffe Server and Giraffe clients.
